## Usage

1.打包docker file
```sh
docker build -t my-python-app .
```

2.運行docker image,啟動 API 服務
```sh
docker run -it --rm -p 5000:5000 --name my-running-app my-python-app
```

3.執行單元測試
```sh
open terminal
pipenv shell
py.test tests/
```
